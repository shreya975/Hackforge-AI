# 🚀 HackForge AI – Multi-Agent Hackathon Builder

HackForge AI is a futuristic multi-agent AI platform that helps developers generate complete hackathon-ready project ideas using Gemini AI.

## ✨ Features

- Idea Generator Agent
- Tech Stack Architect Agent
- System Design Agent
- README Generator Agent
- UI/UX Planner Agent
- Pitch Deck Agent
- Social Media Agent
- Deployment Agent
- Markdown export
- Google Cloud Run ready

## 🛠️ Tech Stack

- Python
- Streamlit
- Gemini API
- Google Cloud Run
- Docker
